Texture Name Converter

### What's the Converter

When converting a VRM file to PMD via PMX, a problem occurs with the texture names.
To resolve this, you need to convert the texture name to an 11-character model number.
This tool is meant to handle that automatically.


### How to use

Create "files" directory and paste your textures into there.
Double-click "main.vbs".
If you want to know the result, open "result.txt".
